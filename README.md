# Familienkalender

Aggregierter Familienkalender als Home-Assistant-Add-on für den Raspberry Pi 5 mit Waveshare-Touch-Display.

Zeigt die Kalender der Familie in einer gemeinsamen Monats-/Wochenansicht:

- **Marina** (Google) — vollständig
- **Kunde** (Google Workspace) — nur Abend- und Mehrtagestermine
- **Firma** (Nextcloud/CalDAV) — nur Abend- und Mehrtagestermine

Dazu: Tages-Symbole (Smileys), Strom-Ansicht (Balkonkraftwerk), Admin-Bereich über die HA-Seitenleiste (Ingress).
